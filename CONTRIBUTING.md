### Who Can Contribute?

## Reporting Issues

### ✅ What You CAN Do:

1. **Report Bugs**: If you find a bug in the official hosted version, please open an issue
2. **Suggest Features**: Feature suggestions are welcome (but not guaranteed to be implemented)
3. **Ask Questions**: Questions about how the software works are acceptable

### ❌ What You CANNOT Do:

1. **Submit Pull Requests**: We do not accept PRs from external contributors
2. **Fork and Modify**: Forking for modification purposes is prohibited
3. **Share Modified Versions**: Creating and sharing derivative works is not allowed

## How to Report a Bug

If you find a bug in the **official hosted version** (not a forked/downloaded copy):

1. Check if the issue already exists
2. Create a new issue with:
   - Clear description of the bug
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots if applicable
   - Browser/device information

**Template**:

```markdown
**Bug Description**
[Clear description]

**Steps to Reproduce**
1. Go to '...'
2. Click on '...'
3. See error

**Expected Behavior**
[What should happen]

**Actual Behavior**
[What actually happens]

**Environment**
- Browser: [e.g., Chrome 120]
- OS: [e.g., Windows 11]
- Device: [e.g., Desktop]
```
